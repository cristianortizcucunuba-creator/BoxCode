Reto 3: El Simulador de Alcancía Digital

Este reto te obligará a mantener el estado de una variable a lo largo del tiempo usando un bucle que no se detiene hasta que se cumpla una condición.

    El objetivo: El usuario define una "meta de ahorro" (por ejemplo, $100). El programa le irá pidiendo que introduzca montos de dinero uno a uno.

    El flujo:

        Cada vez que el usuario ingresa dinero, el programa muestra el total acumulado y cuánto le falta para llegar a la meta.

        El programa sigue pidiendo dinero dentro de un bucle hasta que el total acumulado sea igual o mayor a la meta.

        Al final, celebra con un mensaje: "¡Meta alcanzada! Ahorraste un total de $X".

    Un consejo de programador: No intentes escribir todo el código de un solo golpe. Elige un reto, escribe primero en papel o en comentarios qué pasos necesitas dar (pseudocódigo) y ve programando y probando de a pedacitos.
